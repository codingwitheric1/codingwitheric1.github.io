Getting Started with Python

Here's a step-by-step guide to set up Python on your computer:

Step 1: Download Python

  1. Go to the official Python website: python.org
  2. Click on the Downloads tab
    • The website will suggest the best version for your operating system (Windows, macOS, or Linux)

Step 2: Install Python

For Windows:

  1. Open the downloaded file (python-[version].exe)
  2. Check the box "Add Python to PATH" (important)
  3. Click Install Now and follow the prompts

For macOS:

  1. Open the downloaded .pkg file
  2. Follow the installation instructions
  3. Open the terminal and type python3 --version to confirm installation

For Linux:

  1. Open a terminal and type:
    sudo apt update
    sudo apt install python3
  2. Confirm installation by typing:
    python3 --version

Step 3: Install an IDE or Text Editor

Choose a tool to write and test Python code. Popular options include:

  • IDLE (comes with Python)
  • VS Code: Download from code.visualstudio.com
  • PyCharm: Download from jetbrains.com/pycharm

Step 4: Test Your Python Setup

  1. Open your terminal or IDE
  2. Type:
    print("Hello, World!")
  3. If you see "Hello, World!" on the screen, your setup is complete!
